WeatherWidget
Tweaks


Displays animated weather background on the stock weather widget. (The animated weather backgrounds in the weather app). No noticeable effect on battery life.


Devices with 2GB or more RAM are supported. iPad's not supported. A reboot is required after installation.


Tweak Features

- Randomize weather conditions.
- Select custom weather conditions.
- Works with or without Location Services.
- Supports 3D Touch menu.

Tweak Compatibility

- LockWidgets
- Noctis
