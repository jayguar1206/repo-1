Using Messages & WhatsApp often? ColorizeChat let you add personal touch to your chat app and customize UI elements. If you are looking for bubbles tweak alternative then you have found one. Compatible with iOS 12 & 13

Message Features

  • Gradient cell
  • Gradients bubbles both outgoing and incoming SMS & iMessage
  • Bubbles text color
  • Chat background color
  • Entry bar & field color
  • App drawer bar color
  • Typing bubble and dot color
  • Time stamp & read status text color
  • Table View background color
  • Interaction tint
  • Send button color
  • Swipe action button color
  • Search bar color
  • Navigation bar & large title color
  • Many more features...

WhatsApp Features

  • Gradient Tab Bar
  • Color Tab Bar icons and text
  • Navigation bar & large title color
  • Interaction tint color
  • Chat cell color
  • Color bubbles & text
  • Color chat entry bar and field
  • Color events bubble & text
  • Read status color
  • Color individuals background and cell on Status, Calls, Chats and Settings pages.
  • Swipe action button color
  • Contact list background and footer color
  • Many more features...
