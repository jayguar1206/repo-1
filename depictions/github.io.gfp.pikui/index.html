Pik UI

A Clean & Faded Look to your Phone by Theming your UI. Crafted with Care to Replace the Stock iOS UI with Clean icons with Faded Style Which Feels Very Different and Pleasant to the Eyes.

Pik UI Theme Support Most UI Elements & App Like :

• Most of UI Elements of your Phone

• Phone app + InCall screen UI

• FaceTime + FaceTime InCall UI

• Messages app + Message Chat Bubbles

• Mail app

• Safari app

• Calendar app

• Music app + Music Controls

• System Wide Music Controls

• Lock Screen Toggles icons (Camera and Flashlight)

• Keyboard (Mic, Globe, and Emoji)

• Notes app

• App Store

• Clock app

• Photos app (Including Edit UI & Mark Up UI)

• Share Sheet UI

• Status Bar (DND, Bluetooth, Rotation Lock, Alarm, Location)

Pik UI also Support 3rd Party Applications Like :

• Whatsapp + Whatsapp Chat Bubbles

Pik UI will Support more 3rd Party Application in Updates.

Required Tweaks :

Use Snowboard & Snowboard UI extension from SparkDev repo.

iThemer also supported but I Recommend you to use Snowboard.

For Messages Bubbles :

Some Fixes I Mentioned to get best result of Message Bubbles :

1. BubbleLineRemover (Enable "Disable Mask on images" so the pictures are shown clearly)

2. DopeConvos (Enable "Enable UI theming" option, and "Disable Mask on Images" so the pictures are shown clearly)

3. Tap on UI extension in Snowboard settings and enable “Force Hide Bubble Gradient”

For WhatsApp Bubbles :

If you want to change those colours, Must Use Nord tweak for the best results.

For Keyboard Emoji

follow these steps to Fix Emoji issues :

1. Disable the theme

2. Run iCleaner

3. Enable the theme
