Stock icons suck. Themes are a great way to improve upon Apple's long outdated restrictions.

Glyphy takes it to the next level.

Most themes tend to modify almost all aspects of an app's original icon leaving it almost completely different. While this can look nice, it doesn't tend to feel "native" for the device. Glyphy does the exact opposite. It keeps the good aspects of an icon and perfects all others, making your phone feel like this is how it is meant to look. A perfect balance of beauty and simplicity.

Features:

  • 1820+ icons with more added almost daily
  • 2 user requested icon bg masks for you boomers out there! (jokes guys)
  • UI SWITCHES!
  • MESSAGE BUBBLES!
  • 3 dock styles
  • 6 optional live clock styles
  • 5 optional folder styles
  • Optional settings icons theming WITH tweak icons included ;)
  • Optional StatusBar theming with 3 battery styles
  • 27 alternate icons

Message Bubbles

For the message bubbles, you have several options to remove that pesky line that appears behind text:

  • BubbleLineRemover
  • DopeConvos (Make sure to enable "Enable UI theming" option, and "Disable Mask on Images")
  • Short hold the UI extension in Snowboard settings and enable “Force Hide Bubble Gradient” *Note: if your outgoing text is white and you can’t see it, you need a tweak to color the message bubbles

IMPORTANT NOTE: "SnowBoard Settings Icons Extension", "SnowBoard Preset Extension", and "SnowBoard StatusBar Extension" MUST BE INSTALLED FROM SPARK'S REPO IN ORDER TO THEME THE SETTINGS ICONS AND STATUSBAR! "SnowBoard UI Extension" is needed for the UI Switches and Message bubbles but is also needed for certain parts of the settings theming.
