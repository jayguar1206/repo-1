Divisé
Tweaks
arm64 Dualboots/Tethered Downgrades on device!

Divisé is a Coolbooter-esque app which is capable of arm64 Dualboots and Tethered Downgrades, all from within the app! Checkm8 is used to tether boot the second OS, after dualbooting/tether downgrading.

PyBoot on macOS is required to boot the second OS. Get it here!

Divisé is a fork of Succession, by samg_is_a_ninja, and it's source code is available here!. Feel free to open pull requests/issues for anything!

What's New

    • Fix local IPSW check to use any IPSW located in "/var/mobile/Media/Divise/"
    • Added low storage checks to ensure device has enough space
    • Update labels to have accurate information

Known Issues

    • 10.x support is broken
    • Extracting a locally downloaded IPSW will crash the app, even when the extraction is successful
    • Sometimes, on 12.x and lower, Divisé will will freeze while downloading the chosen IPSW
    • VoiceOver doesn't detect anything to read in the app

    Features

    • Easily dismiss your alerts
      • Simply tap behind the alert to dismiss it!
    • Simplify your alerts
      • Remove the cancel button by enabling 'Remove cancel buttons' in Dismisser's Preferences
    • Simple preferences
      • Dismisser's preference pane is simple and easy to use, and also doesn't require a respring to put your changes into effect

    Notes

    • Dismisser only works on the SpringBoard, supporting apps would be hard as the 'cancelAction' property gets removed, this would cause unexpected behaviour in some apps. Support for it may or may not arrive in the future
    • Dismisser is open source, be sure to check out the code if you're interested.
    • If you have an issue, create an issue here.
    • Dismisser is the successor to DismissAnywhere, I would like to thank the developer for the concept
