Omega

Omega is a custom powerdown menu that aims to achieve both fashion and function. The menu supports a range of different power options and does so in a modern interface. If you don't like the interface, don't worry, I've built Omega with user customization in mind. You will be able to tweak Omega to fit your liking or setup.

Here's what Omega currently includes

  • Customizable layout, behavior, and colors
  • Adapts to light and dark mode, or can be set to a specific one
  • Smooth and modern default presets
  • Entrance and exit animations
  • The ability to reboot user space instead of a total reboot
  • The ability to share and save preferences
  • More to come

Support

Omega supports iOS 12 and 13 as well as a12. It may work on iOS 11 and below as well but is untested so far.

Please feel free to contact me with any questions or concerns. I plan to support Omega for a long time and provide both bugfixes and feature filled updates.

Activation method: First press volume ➕, Then press volume ➖, And then long press the shutdown button to complete the activation
