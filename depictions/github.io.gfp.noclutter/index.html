NoClutter

NoClutter is an all in one tweak dedicated to hiding all the annoyances of iOS. It replaces all those "no" tweaks, like "NoBreadcrumbs" or "NoHomebar".

Open Sourced: https://github.com/QuixThe2nd/NoClutter

Features

Springboard:

  • Hide Homebar
  • Hide Breadcrumbs
  • Hide Scroll Bar
  • Hide Separator Lines
  • Hide Tap Labels

Status Bar:

  • Hide Status Bar
  • Hide Status Bar in CC
  • Hide Time
  • Hide Cellular Bars
  • Hide Wifi
  • Hide Cellular (e.g 4G)
  • Hide Cellular Text (e.g AT&T)
  • Hide Battery
  • Hide VPN
  • Hide Pill

Home Screen:

  • Hide Page Dots
  • Hide Update Dots
  • Hide Beta Dots
  • Hide Dock
  • Hide Dock Background
  • Hide Icon Badge
  • Hide x Button
  • Hide Editing Done Button
  • Hide Widget Footer

Lock Screen:

  • Hide LS Padlock
  • Hide FaceID Glyph
  • Hide Do Not Disturb Banner
  • Hide CC Bar
  • Hide Swipe Text
  • Hide Quick Action Toggles
  • Hide "Notification Center" Text
  • Hide "No Older Notifications" Text
  • Hide Clear Notifications Button

Safari:

  • Hide Close Tab Button
  • Hide Search Background Color
  • Hide Favicon

Folders:

  • Hide Folder Title
  • Hide Folder Background
  • Hide Folder Blur

App Switcher:

  • Hide Header (Icon and Label)
  • Hide Icons

Other:

  • Hide Settings Arrow
  • Bug fixes
