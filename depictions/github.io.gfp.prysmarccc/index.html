1. After installing Prysm Arc CC go to folder location using filza Library/Themes/Prysm Arc CC icons.theme/Bundles you'll find Prysm folder copy prysm folder.
2.Paste and Replace the folder inside folder in Library/Application Support.
3.Adjust toggles from Control Center settings which is with black background icons.
4.Apply Prysm Arc CC using Snowboard or iThemer.
5.Now Respring your device.
