Gina CC for Prysm

Make sure already installed :
- Snowboard
- Snowboard UI Extention
- Prysm
- MagmeEvo

Installation process :
1. After installing Prysm GinaCC go to folder location using filza Library/Themes/Prysm GinaCC icons.theme/Bundles/Prysm, Copy all of the icons.
2. Paste and Replace the files inside folder in Library/Application Support/Prysm/Icons.
3. Adjust toggles from Control Center settings which is with black background icons.
4. Apply Prysm GinaCC using Snowboard or iThemer.
5. Now respring your devices.
