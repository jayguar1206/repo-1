用最漂亮的樣式重新構想CC。
CC精美雕文
這也適用於普通CC
我為MagmaEVO添加了5個禮物（自動安裝在Preset目錄中，安裝)
我在這裡做了一些非常簡單的事情，易於使用。
安裝過程-只需5分鐘
1.安裝Circle CC後，請使用filza Library/Themes/PrysmBigDevyCC（Circle）進入文件夾位置.theme或PrysmSmallDevyCC（Circle）.theme/bundle，您會找到prysm文件夾，複製該文件夾
2.將該文件夾粘貼在Library/Application Support文件夾中。
3.使用SnowBoard或ithemer應用Circle CC，然後註銷設備。
4.從帶有黑色背景圖標的控制中心設置中進行調整
5.現在重新註銷設備。
