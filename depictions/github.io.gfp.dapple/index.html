Supports iOS 11, 12, and 13
It has the ability to change colors of:

  • The WiFi icon.
  • The cellular signals.
  • the text for the time, the carrier label and the battery percentage label.
  • the item/image icons.
  • the battery icon (with 3 individual options, body, fill and pin color.

Experimental features:

  • Set a custom status bar background color.
  • Set a custom navigation bar background color.
  • Set a custom large titles background color.
  • Set a custom UILabel color "system-wide".
  • Set a custom keyboard background color.
  • Force enable the stock dark notifications.
  • Invert the keyboard labels.

Known bugs:
the text option is not separated so that means it applies for all text elements.
the items option is not separated so that means it applies for all the items.
**It doesn't support the old statusbar, so you will need to enable a modern statusbar, such as the iPhoneX,XR,XS,11,11pro and the iPad statusbar.
  • You can now select custom colors for certain items.
