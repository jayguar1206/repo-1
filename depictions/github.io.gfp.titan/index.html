Titan
Tweaks

  • True picture in picture support for all your favorite apps.

    Auto PiP

    Just like on iPad, pip will be started with an animation when exiting an app, opening a notification, etc.

    Toggles/Activator support

    You can set any activator action to toggle pip for those apps that do not include a button. Titan also comes with two similar toggles included if you don’t wish to use activator.

    Smart Positioning

    This automatically tries to figure out where navigation and tab bars are then moves the pip window to not get in the way.
