Gyan Theme

  • 200+ Icons (And will continue to update)

  • Icons Mask
  • StatusBar Theme (Only NiceBarX)
  • Icon Settings Theme
  • 2 Style Badges

You Must Have

  • Snowboard
  • NiceBarX

Installation

  • Select theme icon in snowboard
  • Active statusbar in NiceBarX
  • Select Battery Settings > Enable Customization > icon style select use icons compatible alkaline > choose icon select gyan battery > turn off adaptive icon color
  • Go to System, Carrier, Cellular, WiFi Settings > Enable Custom Cellular Icons > Custom Icons Select Use custom Icons > Icons location select Gyan Cellular > Turn Off adaptive color > icon scale mode select custom ratio > ratio set in 1.5:2.2
  • Scroll dwond and find Custom WiFi > Custom Style select Use Custom Icons > Icon location select Gyan WiFi > Turn Off adaptive color > Icon Scale Mode select custom ratio > set ratio in 1.5:2.1
