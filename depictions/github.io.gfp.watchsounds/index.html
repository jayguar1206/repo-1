Compatible with iOS 11 - 13.

Change the stock notification sounds and vibrations on the Apple Watch.
Currently there are 10 different kinds WatchOS system sounds to choose from.
Most of the sounds and vibrations are never used in the regular WatchOS, but with WatchSounds you can repurpose them to any application you'd like.
Added an option to always forward notifications to the watch, even when the iPhone is unlocked. v1.1.5 - Added Activator integration to this setting!
Also there is an option to replace the default applications sound globally.
Check out the demo video to hear all of the sounds: https://youtu.be/212G6_J7CEQ
List of available sounds:

  • Phone call
  • Text message
  • Calendar event
  • Alarm
  • Digital touch
  • App notification
  • Emergency SOS vibration
  • Breathe
  • Walkie talkie (WatchOS 5+)
  • Health (WatchOS 5+)

Each sound is paired with its own vibration.
In the preference pane there are test buttons to play every one of the available sounds straight on the watch.
Currently there is no option to add your own custom sounds since there is no WatchOS jailbreak, thats why we have to use what is available in WatchOS.
More sounds will be added when WatchOS will be updated with new system sounds.
  • Added Activator integration to the Always Forward To Watch switch.
