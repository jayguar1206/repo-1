This theme is designed to replace your cc icons, weather icons, and fits very well with Zhyr UI.

i will not accept any refunds if thats the excuse!

How to apply?

  • For Prysm you will need to move the icons inside /Library/Themes/Zhyr CC.theme/Folders/Prysm/Icons to /Library/Application Support/Prysm/Icons (But please make a backup of the original icons first)
  • For the stock CC you will need BetterCCIcons13 to get wifi and bluetooth themed! The files are inside the theme folder: /Library/Themes/Zhyr CC.theme/Bundles/BetterCCIconsPro then move the files (not the folder) to: /Library/PreferenceBundles/BetterCCIcons13.bundle, then enable glyph mode in settings.
  • To get the extra toggles you will need FlipConvert, you can find it on Julioverne's repo.
  • I reccomend you use Snowboard and Snowboard UI Extension from Spark's repo.
