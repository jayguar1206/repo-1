Aiir CC Theme

Custom made Control Center Icon Theme. Comes with 2 different icon styles to best match your current theme and preference.
Default icons are white. You will need a tweak like "Magma Evo" to recolour them. Like shown in pictures.
Two Styles Available:

  • Glyph Icons
  • Circle Icons

If you have any issues with icons, please contact me and I will try to resolve as fast and best as possible.

Setup:

To setup everything included you will need either Snowboard or iThemer.
Note: if using Snowboard and you have "Snowboard UI Extension" and its still not theming, disable "Data Collection" in snowboard! Go to Settings> Snowboard> Extensions> Snowboard UI Extension>Settings button>Make sure Data Collection is disabled. That is only used for theme developers.
Snowboard Requirements:

  • Snowboard
  • Snowboard UI Extension

iThemer Requirements:

  • Latest version of iThemer

Optional Tweaks:

  • Prysm - Optional
  • BetterCCiconsPro - Optional
  • Magma Evo - Optional

Instructions - Read Carefully


You must have:

  • CCModules
  • FlipConvert - REPO: https://julioverne.github.io/https://julioverne.github.io/
  • FlipSwitches - REPO: http://rpetri.ch/repo/http://rpetri.ch/repo/
  • SnowBoard
  • SnowBoard UI Extension
  • iThemer (Use either iThemer OR Snowboard + Snowboard UI Extension)

Theming CC Toggles:

  • Open Snowboard > AiirCC > (Style)
  • OR
  • Open iThemer > Enable > AiirCC > (Style)

You do not need both iThemer and Snowboard
Note: Default stock Control Center will not theme your Wifi and Bluetooth icon for that you will need "BetterCCiconsPro" this is the only workaround i know. You do not need this with Prysm as Prysm also solves that issue.
Theming CC Animated Toggles:

  • Navigate to Settings App>Control Centre>Customize Controls.
  • Add the new FlipConvert toggle which has the theme
  • Respring

BetterCCiconsPro:
This is needed if you don't use Prysm, this will be used to theme your Wifi & Bluetooth icons.

  • In filza, navigate to /Library/Themes/AiirCC - (style)/Bundles/BetterCCIcons13
  • Copy the .png icons only inside the folder
  • Navigate to Library/PreferenceBundles/BetterCCIcons13.bundle
  • Paste the icons in that location
  • In BetterCCIcons13 settings, enable Glyph Mode

Magma Evo Theme:

  • Aiir Blue: https://pastebin.com/6rh0iWyChttps://pastebin.com/6rh0iWyC
  • Copy all text > Magma Evo > Import
