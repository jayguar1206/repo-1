Compatible with iOS 11 and 12

Not compatible with iPad.
iKeywi is a tweak which can let you customize the keyboard on your iPhone. The latest version of iKeywi series: iKeywi 4 can not only add a extra row of keys, but also let you flick custom keys like on your iPad!
Features:

  • Adds an extra row to your keyboard
  • Supports all languages (Please check note below)
  • Supports both portrait and landscape mode
  • Compatible with iPhone 5s / 6 / 6 plus / 6s / 6s plus / SE / 7 / 7plus / 8 / 8 plus / X / XR / XS / XS Max
  • ...and more!

Note:

  1. Doesn't support: Kana(かな入力), Korean 3x3 (10키), Pinyin 3x3(十键拼音), Strokes(笔画) and Handwriting(手寫).
  2. Zhuyin(注音) and Thai 5-Row only support Number and Symbol keypad.
  3. This tweak can't work in some apps which need root permission, such as iFile.
  4. Doesn't work for AppStore 3rd party keyboard.
