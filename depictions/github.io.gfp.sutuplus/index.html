SutuPlus（蘇兔微信主題工具）

-

更新兼容微信最新7.0版本

最低兼容微信版本為 6.7.0

對系統版本號沒有特殊的要求

如果微信版本符合但不能用的話

反思一下自己是不是用的子午線或存在其他衝突

-

主要功能如下：

主題列表（主要）

脫離原有主題依賴an加載或封包手動替換的模式

直接使用插件加載美化主題

實現在插件內快捷查看、管理、使用主題

-

直接調用color.css

不用手動根據不同的主題替換color.css了

在搭配主題列表功能的前提下

可以不需要任何多餘操作，切換主題的同時生效color

-

免紅包/卡券圖標替換

同樣不需要手動找到路徑替換了

在製作適配插件主題時

直接和其他圖片素材放置在同一路徑下可直接生效

-

主題配套默認聊天背景

增加了默認的聊天背景文件規範

在主題作者完成了背景製作的前提下

應用主題即可默認幫忙替換聊天背景

-

首頁行背景顏色和透明度配置

避免背景過花看不清楚字

可以通過插件配置透明度和顏色

注：這個功能不會包含聊天列表頁（也就是微信頁）
