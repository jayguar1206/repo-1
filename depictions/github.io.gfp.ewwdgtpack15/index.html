Lockscreen Widgets

EW.WDGTPACK.15

Includes: 5 LS + 2 HS widgets.

LS widgets:

  • Installed to the LockHTML directory: /var/mobile/Library/LockHTML
  • Apply with Xen HTML
  • REQUIRES XENINFO
  • LS1: CLICK THE UPPER ARROW TO VIEW WEATHER, CLICK THE LOWER ARROW TO VIEW BATTERY.

HS widgets:

  • Installed to the SBHTML directory: /var/mobile/Library/SBHTML
  • Apply with Xen HTML
  • REQUIRES XENINFO
  • SB2/IW2: CLICK THE DOTS TO TOGGLE THE WEATHER VIEW.
