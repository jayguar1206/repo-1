Either if you have annoying parents who want to know exactly what you are doing on your phone, or friends who use your phone too often, your privacy is compromised. Not anymore! Let us introduce LendMyPhone 3, a real guest mode for iOS. LendMyPhone 3 is the best guest mode tweak ever, since it allows you to setup multiple users on your device, like on any computer. But wait! It would quite unnecessary if anyone knew they are using a guest user. Yes, it would. Therefore, LendMyPhone 3 never tells that there are multiple users on the device, or which one is currently in use! Just type in the passcode normally, and it will load the home screen based on your configuration. Smart, right? LendMyPhone 3 can control the multiple events on iOS as described down below.


Events


Hidden Apps
• Select which apps to show on the home screen for each user. All other apps will be hidden, like they have never been installed! They will not even show up in the multitasking app switcher!


Control Center
• Select whether a user can open and interact with the control center.


Icon Editing
• Select whether a user can change your pre-set icon layout on their home screen.


Multitasking
• Select whether the user can kill whitelisted apps. That way, you can know exactly what the guest user has been doing or not. Also, you can completely disable the multitasking app switcher for each user individually.


Notification Center
• Select whether a user can open and interact with the notification center.


Spotlight
• Select whether a user can open the spotlight search on the home screen.


You will never have to remove apps anymore when your parents want to know what you are doing on your phone. From now on, you can even tell them the 'passcode'. And you will never ever have to watch over your friends back to know that they are not messing around with your phone. All this, with LendMyPhone 3.
