Nova UI
Got bored using same style over & over try Nova UI it’s Different.
Nova UI concept unique & different with bold & fresh design.
Nova UI Theme Covered below Apps:

  • Completely changing UI
  • Phone & call screen UI
  • FaceTime UI
  • Messages & Message bubbles UI
  • Safari UI
  • Music & Music controls UI
  • Keyboard (Mic, Globe, and Emoji)
  • Notes UI
  • App Store UI
  • Clock UI
  • Photos UI
  • Mail UI
  • Calendar UI
  • Lock screen & control centre music control UI
  • Lock screen toggles icons (Camera and Flashlight)
  • More small UI elements

Nova UI also covered below applications:

  • WhatsApp UI & WhatsApp bubbles UI
  • Cydia UI
  • Zebra UI

Required Tweaks to use Nova UI:

  • Snowboard (Spark’s Repo)
  • Snowboard UI extension
  • iThemer supported but Snowboard is recommended for best result
  • not test with anemone but should be work there

Fix Messages & WhatsApp bubbles:

  • If you want to colour bubbles use Nord tweak.
  • Use BubbleLineRemover for fix line in bubbles.
  • Use DopeConvos for fix pictures in bubbles
  • In Snowboard settings enable “Force Hide Bubble Gradient”

Fix Keyboard Emoji:

  1. Disable the theme
  2. Run iCleaner
  3. Enable the theme
