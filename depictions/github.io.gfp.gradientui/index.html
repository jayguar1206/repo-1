Gradient UI
Don’t just use UI Fill the UI.
Same customization but with nice Twist.
Must Checkout Nova UI, Nova Status Bar & NovaCC by Me.
If you already purchased Nova UI then this is Free Extension for you.
Gradient Theme Covered below Apps :

  • Completely changing UI
  • Phone & call screen UI
  • FaceTime UI
  • Messages & Message bubbles UI
  • Safari UI
  • Music & Music controls UI
  • Keyboard (Mic, Globe, and Emoji)
  • Notes UI
  • App Store UI
  • Clock UI
  • Photos UI
  • Mail UI
  • Calendar UI
  • Lockscreen & control center music control UI
  • Lockscreen toggles icons (Camera and Flashlight)
  • More small UI elements

Gradient UI also covered below applications :

  • Whatsapp UI & Whatsapp bubbles UI
  • Cydia UI
  • Zebra UI
  • Facebook UI
  • Telegram UI
  • DockX

Upcoming Gradient UI updates will support below UI :

  • WeChat UI
  • Filza UI
  • Reddit UI
  • instagram UI
  • Twitter UI
  • WhatsApp Bussiness
  • More features will add from users request also.

Required Tweaks to use Gradient UI :

  • Snowboard (Spark’s Repo)
  • Snowboard UI extension
  • iThemer supported but Snowboard is good for best result

important notes for Messages & Whatsapp bubbles :

  • If you want to colour bubbles use Nord tweak.
  • Use BubbleLineRemover for fix line in bubbles.
  • Use DopeConvos for fix pictures in bubbles
  • In Snowboard settings and enable “Force Hide Bubble Gradient”

important notes for fix Keyboard Emoji :

  1. Disable the theme
  2. Run iCleaner
  3. Enable the theme
